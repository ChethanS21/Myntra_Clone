# Myntra Homepage Clone

A simple frontend clone of the Myntra homepage built using HTML. This project replicates the basic layout and design of the Myntra website, including navigation menus, promotional banners, category sections, and footer information.

## Features

- Responsive navigation bar
- Search bar
- User profile, wishlist, and shopping bag icons
- Promotional banner sections
- Category showcase sections
- Footer with useful links and company information
- Static image-based UI design

## Technologies Used

- HTML5
- Inline CSS
- External Image Resources

## Project Structure

```
project-folder/
│
├── index.html
└── README.md
```

## How to Run

1. Download or clone the repository.

```bash
git clone https://github.com/your-username/myntra-clone.git
```

2. Open the project folder.

3. Open `index.html` in any web browser.

OR

Use VS Code Live Server:

- Install the Live Server extension.
- Right-click on `index.html`.
- Click **Open with Live Server**.
